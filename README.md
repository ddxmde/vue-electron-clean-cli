# cue-electron-clean-cli
基于vue-cli4 和 vue-cli-electron-plugin创建，加入配置项vue.config.js.

## 命令

- 初始化 `npm install`
- 调试vue `npm run serve`
- 编译vue `npm run build`
- 调试electron `npm run electron:serve`
- 编译electron `npm run electron:build`
